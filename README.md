# Ecommerce
**This is a simple e-commerce website built with Django and js**
<br>
<hr>

<h2>Features</h2>
<ul>
<li>Trending products</li>  
<li>Nested Categories</li>

<li>Custom Admin panel </li>
<li>Send Email </li>
<li> Validate phone </li>
<li>Search products </li>

</ul>
